# How to use this site

- Use the search bar or navigation to find a substance.
- Click on a substance to view detailed information, including:
  - Name and other known names
  - Classifications and legal status
  - Reasons for prohibition (with links)
  - Warnings and references
  - Source and more info links
  - When the entry was added/updated
- Use the dark/light mode toggle in the top right for your preferred viewing mode.
