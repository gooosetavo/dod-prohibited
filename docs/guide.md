# User Guide

!!! tip "Quick Start"
    New to the site? Start here for everything you need to know about finding and understanding prohibited substances information.

## How to use this site

!!! info "Navigation"
    - **Search bar**: Type any substance name to quickly find it
    - **Browse menu**: Navigate through all substances alphabetically
    - **Dark/light mode**: Toggle in the top right corner

!!! success "Finding Substances"
    - Use the search bar or navigation to find a substance
    - Click on any substance name to view its detailed page
    - Each substance page includes:
        - **Name and aliases**: Official name plus common alternative names
        - **Classifications**: Legal and regulatory categories
        - **Prohibition reasons**: Why it's banned (with official links)
        - **Warnings**: Safety and health information
        - **References**: Links to official sources and documentation
        - **Timeline**: When the entry was added or last updated

!!! warning "Important Notes"
    - This data is automatically updated daily from official DoD sources
    - Always verify with official sources for the most current regulations
    - This site is for informational purposes only
